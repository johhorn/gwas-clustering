Code for clustering of diseases and traits

This page describes the GitHub repository for the master thesis "Tools for cluster analysis of data from genome-wide association studies: Role of data representation and similarity measures", written by Johanne H. Horn at the research group of Biomedical Informatics (BMI), University of Oslo (UiO).

The majority of the code available in this repository is part of the Genomic HyperBrowser, an open source, web-based software system for comparative genomic analysis. The HyperBrowser is developed and supported by the BMI research group at UiO and the Radium Hospital in Oslo. The online interface can be found here, where the following tools are provided:

Clustering of binary data representations
  • Binary vector clustering
  • Genetic loci clustering
  • Haplotype block clustering
  • Bipartite matching of point tracks
Clustering of continuous vectors
  • Correlation of track values
  • Correlation of overlap with other tracks
Empirical exploration tools
  • Empirical properties of point tracks
  • Empirical properties of LD tracks
  • Print rsids of tracks in GSuite
Create and modify data
  • GSuite liftover
  • Convert GSuite sumstat tracks
  • LD GSuite generator
  • LD track generator

Most of the code provided cannot be run as standalone scripts, as they rely on the HyperBrowser framework, in particular the files of tools, statistics and general helper classes. These tools and statistics provide different implementations for feature representation and comparison of genomic tracks. The tracks represent diseases and traits, and represented by associated variants from Genome-Wide Association Studies (GWAS). In addition to HyperBrowser code, scripts for LD computation for variants, as given by their rsids, is provided.

Code for clustering of diseases and traits

This page describes the GitHub repository for the master thesis "Methods for representation and comparison of diseases and traits", written by Johanne H. Horn at the research group of Biomedical Informatics (BMI), University of Oslo (UiO).

The majority of the implementation is part of the Genomic HyperBrowser, an open source, web-based software system for comparative genomic analysis. The HyperBrowser is developed and supported by the BMI research group at UiO and the Radium Hospital in Oslo. The online interface can be found here, where the following tools are provided: Clustering of binary data representations

  • Binary vector clustering
  • Genetic loci clustering
  • Haplotype block clustering
  • Bipartite matching of point tracks
Clustering of continuous vectors
  • Correlation of track values
  • Correlation of overlap with other tracks
Empirical exploration tools
  • Empirical properties of point tracks
  • Empirical properties of LD tracks
  • Print rsids of tracks in GSuite
Create and modify data
  • GSuite liftover
  • Convert GSuite sumstat tracks
  • LD GSuite generator/
  • LD track generator

The code available in the GitHub repository is the code developed for the purpose of the master thesis. The tools and statistics provide different implementations for feature representation and comparison of genomic tracks. The tracks represent diseases and traits, and represented by associated variants from Genome-Wide Association Studies (GWAS). In addition to HyperBrowser code, scripts for LD expansion of variants, as given by their rsids, is provided.
